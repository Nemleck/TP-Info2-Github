# TP-Info2-Github

## Objectif

> Ce programme comprend un ensemble de plusieurs structures implémentant des formes géométriques.

> Il est possible de les montrer, de récupérer leurs propriétés et de les modifier via une translation, en changeant l'échelle ou en les faisant tourner.

## Utilisation

> Ce programme n'inclut pas d'interface utilisateur : Vous pouvez modifier le main pour utiliser les différentes structures comme souhaité, puis le compiler via la commande `make` et enfin le lancer via la commande `./main`.

## Installation

> Il est possible de télécharger ce programme via github ou avec la commande :
> `https://github.com/Nemleck/TP-Info2-Github`

## Dépendances

> Ce programme utilise c++23 et est compilé via un makefile avec g++. <br />
> Il utilise également CTurtle mais ce module est inclus dans le projet.
